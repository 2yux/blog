	- 编程：
		- `git reset --soft HEAD^`：
			- `--soft`：撤销 commit，不撤销 `git add .` 操作，不删除改动的代码
			- `--mixed `：撤销 commit，撤销 `git add .` 操作，不删除改动的代码
			- `--hard`：撤销 commit，撤销 `git add .` 操作，删除改动的代码，危险操作
			- 如果进行了2次 commit，都想撤销，使用：`HEAD~2`
